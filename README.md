Setup instructions: 
1. Make sure python 3 is installed on your machine.
2. Use git clone https://github.com/sabeeh06/fitness-ai-catalog to download the application.
3. Make sure frontend.py and fitness_catalog.csv are in the same folder.
4. Open the folder in an IDE and run frontend.py to start the application. OR change to the fitness-ai-catalog in the terminal and run the command 'python3 frontend.py'.
